## Deployment

To deploy this project run

```bash
  git clone https://github.com/oglis22/Analytic
  cd Analytic
  npm i
  npm start
```


## Tech Stack

**Client:** EJS

**Server:** Node, Express, body-parser, ChartJS, cookie-parser, discord.js, ejs
